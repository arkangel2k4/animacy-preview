M1.5 — the renderer swap
Same M1 voice server, untouched — only the renderer changed: the character is now a standard VRM (pro rig, springbone physics, spec visemes), body tags play real mocap where clips exist and hand-tuned 3D gestures elsewhere (full arm freedom — no 2D tearing), Cartesia word timestamps drive aa/ih/ou/ee/oh mouth shapes, face tags map to VRM expression presets, and [scene:] tags swap the painted backdrops. Demo character = VRM Consortium sample — replace VRM_URL with your hero model. Mocap coverage grows with the Mixamo batch; unmapped tags degrade gracefully to gestures.
